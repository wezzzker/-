# НПЦ ИРС

## Начало работы

### Server -->

Откройте терминал в корневой папке проекта и пропишите следующее:

> cd server
> npm install
> npm run dev

### Client -->

После инициализации сервера в том же теерминале пропишите следующее:

> cd ../  
> cd client
> npm install
> npm run dev

### Посмотреть приложение можно по адрессу http://localhost:5173/
